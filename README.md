 📚 Sistema de Biblioteca em Java

 📖 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar os principais conceitos de Programação Orientada a Objetos (POO) utilizando Java.

O sistema funciona em modo console e permite realizar o gerenciamento básico de uma biblioteca, simulando o cadastro de livros e usuários, além do controle de empréstimos.

Durante o desenvolvimento, foram aplicados conceitos fundamentais de orientação a objetos e boas práticas de organização do código.

---

 Funcionalidades

* ✅ Cadastro de livros
* ✅ Listagem de livros
* ✅ Busca de livros por título
* ✅ Cadastro de usuários
* ✅ Listagem de usuários
* ✅ Empréstimo de livros
* ✅ Controle de disponibilidade do livro

---

Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Collections (`ArrayList`)
* Scanner para entrada de dados
* Estruturas de repetição (`for`)
* Estruturas condicionais (`if`, `switch`)
* Encapsulamento
* Métodos e organização por responsabilidades

---

Estrutura do projeto

* **Biblioteca** → Responsável pelo gerenciamento do sistema e menu principal.
* **Livro** → Representa os livros cadastrados na biblioteca.
* **Usuario** → Representa os usuários cadastrados.
* **Main** → Inicializa a aplicação.

---

 Como executar

1. Clone este repositório.
2. Abra o projeto em uma IDE Java (IntelliJ IDEA, Eclipse ou VS Code).
3. Execute a classe `Main`.
4. Utilize o menu para navegar pelas funcionalidades do sistema.
